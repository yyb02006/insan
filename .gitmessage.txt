# 커밋 타입 리스트
# Feat : 새로운 기능, 요소 추가
# Fix : 버그 수정
# !HOTFIX : 급한 버그 핫픽스
# Design : CSS 등 UI 변경
# Comment : 주석 변경
# Refactor : 코드 리팩토링
# Style : 기능적 수정 없이 코드 포맷만 변경
# Docs : 문서 수정
# Test : 테스트 코드 추가
# Chore : 빌드 프로세스 수정 혹은 패키지 매니저 수정사항
# Rename : 파일 혹은 폴더명의 수정이나 위치 변경
# Remove : 파일 삭제
################

# Fixes : 이슈 수정중
# Resolves : 이슈 수정 완료
# Ref : 참고 이슈
# Related to : 해결되지 않은 관련 이슈
